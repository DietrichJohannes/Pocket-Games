# 🎮 Pocket Games

Willkommen bei **Pocket Games** – einer Sammlung selbst programmierter Retro-Spieleklassiker!  
Hier findest du beliebte Arcade-Games wie **Flappy Bird**, **Tetris** und etc.

## 🔧 Voraussetzungen

- Windows 10/11
- Keine zusätzliche Software erforderlich

## 🚀 Ziel des Projekts

Ich entwickle diese Spiele als Lernprojekt, um meine Programmierfähigkeiten (C++, WinAPI, evtl. SDL/Allegro) zu verbessern und gleichzeitig nostalgische Spiele aufleben zu lassen.  
Feedback, Ideen und Verbesserungen sind jederzeit willkommen!
